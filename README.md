# Каталог фильмов и сериалов

Веб-приложение на Flask для просмотра, добавления и управления фильмами и сериалами.  
Включает REST API и веб-интерфейс с авторизацией и разграничением прав.

## Функционал

- Просмотр фильмов и сериалов (доступно всем)
- Добавление, редактирование и удаление записей (только авторизованные)
- Удаление — только для администратора
- RESTful API для управления данными
- Swagger-документация API (`/apidocs`)
- Русскоязычный интерфейс

## Технологии

- Python
- Flask
- Flask-SQLAlchemy (ORM)
- Flask-Login (аутентификация)
- Flask-WTF (формы и CSRF-защита)
- Jinja2 (шаблонизация)
- Bootstrap 5 (стилизация)
- SQLite (база данных)
- flasgger (Swagger UI)
- pytest (тестирование)

## Установка и запуск

1. Установите зависимости:
   pip install -r requirements.txt
2. Запустите приложение:
   python run.py
3. Откройте в браузере:
   Веб-интерфейс: http://127.0.0.1:5000
   Документация API: http://127.0.0.1:5000/apidocs

Учётные данные администратора
